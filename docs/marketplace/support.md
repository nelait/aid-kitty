## Common Issues

### Sign-In Problems
- **Microsoft SSO**: Ensure your organization allows third-party app access. Contact your IT admin if you see "Need admin approval."
- **Email/Password**: Use the "Forgot Password" flow or contact support for a reset.

### AI Generation
- **No providers available**: Add at least one AI provider API key in Settings.
- **Generation timeout**: Large documents may take up to 60 seconds. Try a different AI provider if issues persist.

### Subscription
- **Subscription not active**: After purchasing on AppSource, allow up to 5 minutes for activation. If still inactive, contact support.
- **Plan changes**: Managed through Microsoft AppSource. Changes take effect at the next billing cycle.
